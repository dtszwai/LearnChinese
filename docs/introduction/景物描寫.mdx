所謂的描寫文，就是對人、景物描繪的文章，把人物、景物、事件的特徵描繪表達出來。

:::tip 景物描寫

景物包含時間季節、天氣氣象、山林原野、江河湖海、動植物、建築物、交通工具等。  
具體描寫自然風光，營造一種氣氛，烘托人物的情感和思想。

:::
  
常見的景物描寫方法有：

1. 感官描寫
2. 動態描寫／靜態描寫
3. 隨時推移法
4. 定點描寫
5. 步移法
6. 多角度描寫

## 感官描寫

運用不同的感官、感覺去描寫景物或事物的寫作手法，如視覺、聽覺、嗅覺、味覺、觸覺等，令所描寫的事物更具體傳神。

1. 👀 **視覺描寫**：  
   如艾雯〈綠水三千〉`寫旭日初升的視覺震撼`：  
   初升的太陽在霧雰裡突圍著，射出一支支金箭，穿破了逐漸輕薄的霧層 ── 突然間一個黃澄澄，光燦燦的太暢脫穎而出，瞬時間雲消霧散，祇見遠山凝黛，叢樹縈翠，一片金光照得潭水閃閃發亮，綠得似萬頃蛟潔明淨不沾半點塵瑕的綠玻璃，竟然是一個透澈晶瑩的世界！

2. 👂 **聽覺描寫**：  
   如吳繼文〈天河撩亂〉`對風聲的描摹`：  
   他靜靜躺著，庭院中的樹葉在夜風中發出沙沙沙的聲音，窗格子輕輕震響。然後他開始聽見風中隱約傳來一些低聲絮語，有時像是獨白，有時又像同時有兩三人在對話。

3. 👃 **嗅覺描寫**：  
   如季季〈一九八四年三月〉`對花香的描述`：  
   夜風把農會旁那叢夜香花的香氣，一波波送來親吻我的臉頰，我的鼻孔。我於是站定了，讓那香氣深深的，一直親吻到我的心底。

4. 👄 **味覺描寫**：  
   如莊因〈花花果果〉`對甘蔗滋味的描述`：  
   甘蔗，那麼強烈地攫住了我的注意。我含笑取了一節逕自放入口中，棲遲異域淡涸了的口腔，就在鬆動了的齒牙奮力咬拚之下，讓一口濃蜜甘例的蔗汁滋潤了齦根，彷彿瓊漿在口的美盛快意，直落下心頭。

5. ✋ **觸覺描寫**：  
   如林文月〈不見瑠公圳〉`對冬夜寒風刺骨的描述`：  
   冬夜，則將衣領豎起，縮著脖子，把凍僵的雙手插入口袋裡，踹踏著冰冷的馬路鑽進布篷下，炒菜煮食的火光，驅走了新生南路上冷空氣，讓沉默的瑠公圳兀自承受亞熱帶冷鋒襲來寒風冷雨。

## 靜態描寫

對靜止不動的事物進行描寫就是靜態描寫，讓讀者掌握景物的靜態美。

靜態描寫運用在人物描寫中，多寫人物外貌、神態等。

至於景物描寫，可以側重描寫景物整體的樣子、位置、顏色、形狀、細部的紋路變化等。

```txt title="靜態描寫例子"
1. 在大河的下游，水流緩慢，岸邊聳立垂直的山壁，十分壯觀。遠遠望去，簡直就像一座巨大無比的天然屏風。

2. 在博物館門前樹立一尊潔白的大理石古人雕像。他戴冠，頭部微微低垂，似在靜靜地思考。他一把漂亮的鬍子飄在胸前，雙手靠在背後，身上穿寬大的衣服，顯得從容而有風度。

3. 竹籬下，有一叢淡淡的野菊花，花朵上還殘留幾滴早晨的露水。旁邊有一棵可愛的小樹苗，葉子綠得如同潤的玉石，樹苗的周圍長滿了亂七八糟的小草。
```

## 動態描寫

對活動變化着的事物進行的描寫，凸顯景物的動態美，增加動感。

動態描寫運用在人物描寫中，多寫人物的談吐、顰笑、行動、待人接物的儀態，可以反映出人物的精神風貌和內心世界。

在描寫景物時可以側重於描寫景物的變化：例如陽光的變化、水流、雲朵、風中的葉子、遊戲中的孩子等等。

```txt title="動態描寫例子"
1. 大河的上游，水流較急，不停地捲着浪花奔湧向前，好像不知疲倦的樣子。它不時翻起一個巨浪，把一隻小小的漁船舉起來，再一個巨浪，又把船擊下去。

2. 剛開始所有的山巒都籠罩在濃濃的霧氣當中，陽光漸漸地照射開來，遠山開始揭開面紗，若隱若現，近處的房舍沐浴金光，花朵也綻放開紅紅的笑臉，遠看就像織出一片燦爛的雲霞。

3. 游泳池中，充滿了人們的歡聲笑語。一個穿紅色泳衣的小女孩在水中一起一伏，在努力練習蛙泳，旁邊的教練在不停地鼓勵她。有的小朋友在水中玩水球，掀起一片片晶瑩的浪花，還有些在池邊的躺椅中舒服地享受日光浴。
```

## 隨時推移

依照時間的變化來描寫同一景物，呈現在不同時間景物的不同面貌，令描寫對象更豐富全面。

```txt title="窗外"
　　窗是屋子的眼睛，打開窗，室內室外便連接起來，美麗的景物也走進來，清新的空氣也流進來。所以，我喜歡在樓上，打開窗，欣賞窗外的世界。

　　當清晨，窗外籠罩著層層的晨霧，但馬路上已有早起的人群，在慢跑，做運動。當晨霧漸散去，可以看到遠方高樓，好像仍在夢中。而居家附近，麻雀已來報早安，吱吱喳喳的在付近跳躍，也跳近我家的陽臺，與我家養的鳥兒，合唱著輕快的早安曲。聲音輕脆，體態輕盈活潑。

　　黃昏時，窗外可以看到滿天的彩霞，像紅色的彩帶，像待嫁姑娘的紅暈，說不上的美麗。更可以看到飛鳥，或成群在天空飛，也許是鴿子吧！或是歸鳥，正在回家的路上。那種飛翔的姿態，可以算是高水準的飛行家。而巷口開始有小孩出來嬉戲，活潑又有趣。

　　夜晚，當讀書讀得有些倦怠，看看窗外的天空，星星在閃爍。雖然無法像在高山上，可以飽攬整個星空，但至少也可以對著星辰，或遐思，或想像，樂趣也無窮呀！有時，從遠方也會傳來蟲吟，甚至雄壯的蛙鳴，讓我也沾染城市中幾分的鄉情呢！有時，城市一角放起燄火，從窗戶，也可飽欖那燦燦多變的燄火，騰空而綻放，為形式化的生活增添幾分詩情與美感。

　　別緊緊關閉窗戶，打開窗，接觸窗外的世界。尤其是要打開心靈的窗，接觸廣大的世界，接觸活潑多彩的人生，讓自己活得愉快美好！
```
