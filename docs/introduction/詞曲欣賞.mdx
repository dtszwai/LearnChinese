## 元曲

### 元曲的興起

- 宋詞不能歌唱，形式更日漸僵化，促使元曲的興起
- 北方外族勢力南侵，胡樂傳入中原，但原來典雅綺麗的詞不能與之配合，要依照胡樂另寫新曲，元曲遂興起
- 文人致力於受人歡迎的戲曲創作才有出路，所以元曲發展迅速
- 元曲內容多反映社會黑暗，漢人作家往往在曲中寄托對百姓的同情
- 元曲四大家：關漢卿、馬致遠、白樸、鄭光祖

＊散曲有「曲牌」。即曲子的名稱，它規定了曲子的句數、字數、平仄、押韻等格式。同一個曲牌可以有不同的題目。

＊元曲曲牌雖有一定字數句式，但因演唱需要，作者可增加字數，稱為「襯」字（用小字標寫）

## 鞏固知識

下列關於「曲」的敘述，何者錯誤？

1. 為元代文學主流，含散曲與雜劇
2. 散曲多出文人之手，風格華美典麗
3. 曲牌又稱曲調，為一首曲的固定格式
4. 套曲乃聯綴同一宮調的若干曲牌而成
