<div className="main prose">

<Annotate title='即北京，民國初年建都南京，北京改稱北平。1949 年中華人民共和國成立，以北平為首都，復稱北京。'>北平</Annotate>，確是一個值得留戀的地方，每遇到冬天的風雪，我腦中必浮現了一個風雪中的北平。

當我住在北平那一年，有一天我早上起來，往城南訪一個朋友，去時太陽本已有些<Annotate title="北方土話颳大風之前，日光微弱，太陽周圍有一層朦朧的光圈，看起來好像長了白一樣，稱為發毛。">發毛</Annotate>；一點鐘左右，起身回家時，北風正一陣緊似一陣的颳着，天也黑將下來。我滿望到了<Annotate title='粵 [羅]，[lo4]。'>騾</Annotate>馬市大街，可以雇車；不料走到那平日「<Annotate title="肩膊和肩膊相摩，車輪和車輪相撞。形容人車絡繹不絕，熱鬧擁擠。轂，車輪的中心部分，借代為車輪。粵 [谷]，[guk7]。">肩摩轂擊</Annotate>」的大街，一看，完全換了樣，不要說車，連行人也沒有。原來那些「<Annotate title="形容行人來往眾多，非常熱鬧。攘，粵 [羊]，[joeng4]。">熙來攘往</Annotate>」的人都受了風的威脅，躲在家裏了，只剩我一人硬頭皮和風在奪路。風真大，我這時又逆着風向走，走上了三步，倒退了兩步，這可沒有法了。忽然想着<Annotate title="位於浙江桐廬富春江，又稱七里灘、七里瀨。傳說是東漢嚴光隱居垂釣的地方。">嚴陵灘</Annotate>上那些背着<Annotate title="拉船前進的粗繩。粵 [牽]，[hin1]。">縴</Annotate>，逆水行舟的縴夫，我便往前斜着身子，學他們的走法，一步一步的和風<Annotate title='豁出去。粵 [判]，[pun3]。'>拚</Annotate>命。這樣，果然好了些。但是一路上棋子般大的泥塊，夾着粉末似的細沙，被風捲起來，儘往我臉上打。眼睛裏、鼻子裏、耳裏、嘴裏，甚至牙齒縫兒裏，滿是沙；不必說，臉上着了風帶着的泥塊，又是疼，又是冷，愈冷就愈疼。不好，到了宣武門的城門洞了。這時候，風給城門洞一逼緊，力量益發大，我真有點「寸步難移」。我只好掉轉頭，背着風，採用「沉着應戰」的方略——乘它一休息，我就回身趕上進佔了幾步；它一來，我又掉轉頭，背着它，靜待着。照這樣的方略，支撐了三次，總算進了城。於是再用縴夫的走法，苦苦的奪路到了家。

當晚我因和風奮鬥之餘，疲乏已極，倒頭便睡。一覺醒來，風聲沒有了，紙窗上大放晴光。我心裏一喜，趕緊起來，捲上窗簾一望，呀！原來是一天大雪，竟在一個晚上，不聲不響的，將我家那院子「<Annotate title="白粉裝飾的，白玉雕成的。這裏比喻院中的景物被白雪覆蓋起來的樣子。">粉妝玉琢</Annotate>」起來。

我住的地方，叫大石作，左景山而右北海，離兩處都不過「一箭之地」。我那時忽發雅興，想揀一高處看看北平的雪景，當下揀定了北海中的白塔。我靠手杖的幫助，一氣跑上了塔的絕頂；四面眺望，整個北平內城就收在眼底，給那光明的、皎潔的、靜默的雪，瀰漫着，懷抱着，和昨天飛沙走石，漫天黑暗，大風中的北平一比較，我真不信是在同一個世界之內——昨天是一個煙塵蔽天的北平，今天是天地空無一點塵了。俯視足下的五龍亭、漪瀾堂一帶，盡成了「<Annotate title='形容精美華麗的樓閣。瓊，粵 [鯨]，[king4]。'>瓊樓玉宇</Annotate>」，而我已<Annotate title='縱身向上。聳，粵 [慫]，[sung2]。'>聳身</Annotate>在瓊樓玉宇之巔。這時候，我真覺有些「<Annotate title='高地之寒意令人無法承受。勝：禁得起、承受得了。粵 [升]，[sing1]。'>高處不勝寒</Annotate>」了。

</div>

1. 北平刮大風前，有甚麼預兆？

    <Task>

    北平刮大風前，日光微弱，太陽周圍有一層朦朧的光圈，好像長了白毛一樣。

    </Task>

2.  刮大風時的北平，街上行人的情形怎樣？街上的風勢怎樣？城門洞裏的風勢又怎樣？

    <Task>

    (1) 刮大風時的北平，街上的情形與平日不同，人們都受了風的威脅，躲在家裏，街上沒有了車，也沒有了行人。

    (2) 街上的風勢很大，人在街上走，走上三步，便被風吹後兩步，風刮起的泥沙打在行人的臉上，使人覺得又疼又冷。

    (3) 在城門洞裏，風勢被城門洞一逼緊，力量更大，使人寸步難移。

    </Task>

3.  作者在回家途中和風奪路，經歷了三個階段，試把它們分別寫出來。

    <Task>

    (1) 第一個階段：只是硬著頭皮地走，走上了三步，倒退了兩步，毫無辦法。

    (2) 第二個階段：想起嚴陵灘上那些背著縴，逆水行舟的縴夫，於是往前斜著身子，學著縴夫的走法，一步一步的和風拚命。

    (3) 第三個階段：穿過宣武門的城門洞時，風被城門洞逼緊，風力更大，使他「寸步難移」，只好採用「沈著應戰」的方法：風來，他便掉轉頭，背著風；風一歇，他便回身趕上進佔幾步。用這方略，支撐了三次，才進了城。

    </Task>

4.  作者選了哪一個地點來觀賞雪景？他為甚麼選擇那個地點？

    <Task>

    作者選了北海的白塔來觀賞雪景。因為北海離他的家不遠，而且白塔是個高處，可登塔頂，居高臨下，觀賞整個北平內城的雪景。

    </Task>

5.  作者所見的北平雪景是怎樣的？他欣賞雪景時有甚麼感覺？

    <Task>

    (1) 作者看見整個北平內城被光明的、皎潔的、靜默的雪覆蓋，變成空無一點塵，而北海一帶更變成了瓊樓玉宇。

    (2) 作者欣賞雪景時，有聳身在瓊樓玉宇之巔，高處不勝寒的感覺。

    </Task>

1.  作者描寫的重點是刮風時的騾馬市大街，為甚麼又寫平日的騾馬市大街？這是甚麼手法？（提示：把兩種相反的東西放在一起而產生效果的修辭手法。）

    <Task>

    (1) 因為他要用騾馬市大街平日熙來攘往的情形，襯托出刮大風時街上連行人都沒有的情形。

    (2) 這種修辭手法叫[襯托](/cheatsheet#修辭手法-襯托)。這裏用熱鬧來襯托冷清，叫做反襯。

    </Task>

2.  作者用了甚麼方法來描寫北平刮大風的情形？（提示：留意文章直接寫風的部分和間接寫風的部分。）

    <Task>

    作者用了直接描寫和間接描寫的方法來描寫北平刮大風的情形。如文中「風真大，我這時又逆著風向走，走上了三步，倒退了兩步」、「風給城門洞一逼緊，力量益發大」，便是直接描寫。而寫騾馬市大街連行人也沒有，寫「眼睛裏、鼻子裏、耳裏、嘴裏，甚至牙齒縫兒裏，滿是沙，……臉上著了風帶著的泥塊，又是疼，又是冷，越冷就越疼」，便是間接描寫。

    </Task>

3.  作者怎樣從描寫刮風的北平過渡到描寫鋪滿白雪的北平？

    <Task>

    作者用了一個小段落，寫他和風奮鬥之後，疲乏已極，倒頭便睡。一覺醒來，風聲沒有了，整個北平城已鋪滿白雪了。這個段落很自然地從描寫刮風的北平過渡到描寫鋪滿白雪的北平。
   
    </Task>

4.  作者欣賞雪景時，觀察點有移動嗎？觀察的角度有甚麼變化？

    <Task>

    (1) 作者欣賞雪景時，觀察點沒有移動，由始至終，都在北海 白塔上。

    (2) 作者觀察的角度有變化，他先「四面眺望」，轉了三百六十度來觀賞整個北平內城的雪景，然後換了一個俯角度，欣賞北海中的雪景。

    </Task>
